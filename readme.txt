== Installation ==

= Manual Installation =

1. Upload the entire `/shopimint-api` directory to the `/wp-content/plugins/` directory.
1. Activate 'Shopimint API' through the 'Plugins' menu in WordPress.

= Better Installation =

1. Go to Plugins > Add New in your WordPress admin and search for 'Shopimint API'.
1. Click Install.
